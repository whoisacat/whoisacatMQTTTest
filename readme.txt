Написать приложение основанное на клиента PAHO для MQTT.
Брокера для тестирования можно создать на этой (https://www.cloudmqtt.com/) платформе.

Необходимый функционал:
а)  Реквизиты для подключения к брокеру:
– доменное имя и ip-адрес;
– порт;
– имя пользователя;
– пароль;
б) Поддержать подписку на один топик. Топик должен задаваться пользователем и поддерживать wildcard(‘#’ и ‘+’);
в) Поддерживать вывод данных с топика на который осуществлена подписка на экран;
г) Поддержать публикацию данных. Топик и публикуемые данные должны задаваться пользователем.
